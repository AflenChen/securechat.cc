## 推送代码的两种方式

### 方式 1: 使用 SSH（推荐）
如果已配置 SSH key：
```bash
git push -u origin main
```

如果未配置 SSH key，需要：
1. 生成 SSH key: `ssh-keygen -t ed25519 -C "AflenChen@users.noreply.github.com"`
2. 添加到 SSH agent: `ssh-add ~/.ssh/id_ed25519`
3. 将公钥添加到 GitHub: https://github.com/settings/keys

### 方式 2: 使用 Personal Access Token
1. 切换到 HTTPS: `git remote set-url origin https://github.com/AflenChen/securechat.cc.git`
2. 使用 token 推送: `git push -u origin main`
