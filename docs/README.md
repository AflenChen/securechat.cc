# Secure Chat Documentation

## Project Overview

Secure Chat is an AI-powered chat helper that provides contextual reply suggestions with 20+ personas. All processing happens locally in the browser for complete privacy.

## Required Assets

### Icons (Need to be generated)
Place the following icon files in `/assets/icons/`:

- `favicon.png` (32x32 or 64x64)
- `apple-touch-icon.png` (180x180)
- `icon-192x192.png` (192x192) - PWA icon
- `icon-512x512.png` (512x512) - PWA icon
- `icon.svg` - SVG source file

### Images (Need to be generated)
Place the following image file in `/assets/images/`:

- `og-image.jpg` (1200x630) - Open Graph sharing image

### Icon Theme
The icon should represent:
- Chat/messaging
- Security/privacy (lock or shield)
- AI/assistant (can be abstract or representational)

### OG Image Theme
The Open Graph image should show:
- "Secure Chat" or "AI Chat Helper" branding
- Chat/messaging visual elements
- Privacy/security indicators
- Modern, professional design
